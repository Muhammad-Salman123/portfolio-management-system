\# Portfolio Management System



A full-stack Portfolio Management System built with the MERN stack (MongoDB, Express.js, React, Node.js).



\## Features



\- User Authentication (Register/Login/Logout)

\- Profile Management (Personal Info, About, Contact)

\- Skills Management (Add/Edit/Delete Skills)

\- Projects Management (Add/Edit/Delete Projects)

\- Responsive React Frontend



\## Tech Stack



\### Backend

\- Node.js

\- Express.js

\- MongoDB with Mongoose

\- JWT Authentication

\- bcrypt for password hashing



\### Frontend

\- React

\- React Router for navigation

\- Axios for API calls



\## Installation



\### Prerequisites

\- Node.js installed

\- MongoDB installed locally



\### Backend Setup



```bash

cd backend

npm install

node server.js


### Frontend Setup

```bash

cd frontend

npm install

npm start
```
